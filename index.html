US Soccer Lineup Builder

Welcome to the US Soccer Lineup Builder!

Directions:

1. Use the buttons at the top to select whether you'd like to build your squad for the US men's national team or the US Women's national team.

2. Use the formation buttons to select the formation you'd like to build your squad in. If you change formations, your player selections will be cleared!

3. Use the drop-down menus to select a player for each position OR type a name in the text box if the player you want isn't listed. You should select 26 players: two for each outfield position, 3 goalkeepers, and 3 other players of any position.

4. Press the "Submit" button to save your data to the server. If you want to start from scratch, press the "Clear" button to get a blank page.
